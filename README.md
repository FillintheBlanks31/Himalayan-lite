# Himalayan-lite

这是一个模仿喜马拉雅lite的微信小程序demo,完全使用微信小程序原生开发，没有使用组件。所以这是一个初级项目，非常的适合微信小程序开发新手，如果你有 html css js 前端基础那么你将很快上手<br/><br/>


> 大神路过勿喷！

代码完全免费开源，看得上可以自取哦（start不start看你的心情🤔）

> 在此项目中部分icon资源和ui使用了colorUI
